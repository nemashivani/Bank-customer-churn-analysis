# 📊 Bank Customer Churn Analysis

## 🔍 Project Overview
This project focuses on analyzing bank customer data to identify patterns and reasons behind customer churn. The goal is to help businesses improve customer retention by understanding key factors influencing churn.

---

## 📁 Dataset
- Raw dataset containing customer information such as:
  - Customer ID
  - Credit Score
  - country
  - Age
  - Gender
  - Account balance
  - Tenure
  - Product number
  - Credit card
  - Estimated salary
  - Active status
  - Churn (Exited)

---

## 🧹 Data Cleaning & Transformation (Power Query)
The raw dataset was cleaned and transformed using Power Query Editor:

- Removed null and duplicate values  
- Corrected data types (numbers, text)  
- Renamed columns for better understanding  
- Created new add columns (if applicable)  
- Filtered irrelevant data  

---

## 📊 Data Visualization (Power BI)
An interactive dashboard was created to analyze customer churn:

### Key Visuals:
- Total Customers vs Churn Customers  
- Churn Rate KPI  
- Churn by Gender  
- Churn by Geography  
- Churn by Amount Balance & Tenure 
- Churn by credit_card_status & credit_score_id
- Active vs Inactive Customers  

---

## 📈 Key Insights
- Customers with lower tenure are more likely to churn  
- Inactive customers show higher churn rate  
- Certain regions have higher churn compared to others  
- Customers with fewer products tend to leave more  

---

## 🛠 Tools Used
- Power BI (Power Query + Dashboard)  
- Excel (optional preprocessing)  

---

## 📌 Conclusion
This project demonstrates how data analysis can help identify customer behavior patterns and support decision-making for improving customer retention strategies.

---

## Business Problem
Why are customers leaving the bank and how can churn be reduced?
